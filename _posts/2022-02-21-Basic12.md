## 문제
/* 2. 제곱 구현 */

두 수 x, y 를 입력 받아 x의 y 제곱 값을 반환해주는 함수를 구현하시오.
제곱 연산자(**)는 사용하지 않는다.

## 나의 풀이
```javascript

function answer(x, y) {
  let result = 1;

  for (let i = 0; i < y; i++){
    result *= x
    //result = 1 이니까 1 * 2 = 2(result), 2*2 = 4(result)
    //이런 방식으로 제곱을 할수있다 (그냥 제곱연산자 쓰고싶다.)
  }
  // …

  // 코드 구현 종료 영역

  return result;
}

/* main code */
let input = [
  // TC: 1
  [2, 3],
  // TC: 2
  [4, 6],
  // TC: 3
  [1, 100],
];

for (let i = 0; i < input.length; i++) {
  console.log(`#${i + 1} ${answer(input[i][0], input[i][1])}`);
}

```
***

## 해답
```javascript
/* 2. 제곱 구현 */

/* user code */
function answer(x, y) {
  let result = 1;

  for (let i = 0; i < y; i++) {
    result *= x;
  }

  return result;
}

/* main code */
let input = [
  // TC: 1
  [2, 3],
  // TC: 2
  [4, 6],
  // TC: 3
  [1, 100],
];

for (let i = 0; i < input.length; i++) {
  console.log(`#${i + 1} ${answer(input[i][0], input[i][1])}`);
}

```

## 부족한 점

아직 for문 활용과 연산자의 활용이 부족함.
